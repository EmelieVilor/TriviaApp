# TriviaApp
## Trivia App | React Native | TypeScript

## Description
A trivia game made in React Native, using Expo go and using https://the-trivia-api.com/ for trivia-questions. 

## Rules of the game: 
 Enter your name, and start quizzing. Pick a category, or try them all. Pick the correct answer and score points. Compete with your friends to see who scores the highest. 

## Purpose
Purpose of this project is to build an app using React Native, Expo and TypeScript. The app should be structured with expo router, using api calls and react hooks.
